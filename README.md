## How to Run

1. Clone the repository  
```
git clone https://github.com/yadavallianusha/MYDAILYWORK.git
```

2. Navigate to the project folder  
```
cd MYDAILYWORK
```

3. Open any task folder  
Example:
```
cd Task1_TodoList
```

4. Run the Python file  
Example:
```
python todo.py
```
