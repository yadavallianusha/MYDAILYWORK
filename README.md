## Requirements
- Python 3.x

## How to Run
1. Clone the repository  
   `git clone https://github.com/yadavallianusha/MYDAILYWORK.git`

2. Open any task folder  
   Example: `Task1_TodoList`

3. Run the Python file  
   `python filename.py`  
   (Example: `python todo.py`)
